explicit

Parameterised constructor with initial capacity hint.

Parameters
nDesired capacity hint; the internal capacity is reserved as n*2. If n <= 0 it defaults to 5 before doubling.

Allows callers to specify expected usage, reducing the number of reallocation events. Size is 0 after construction regardless of n.
