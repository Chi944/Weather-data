Detailed Description

Unit tests for Map<K,V> — Test Plan tests 1-10.

Tests cover all four public methods of Map (version 2.0): operator[] — insert-or-access getPtr() — safe read; nullptr for absent key size() — entry count; size()==0 replaces the old isEmpty() getKey(i) — key at index; combined with getPtr replaces old getValue(i)

Previously tested methods that have been removed in version 2.0: contains() — replaced by getPtr(key) != nullptr isEmpty() — replaced by size() == 0 remove() — never called in the application; removed from interface getValue(i) — replaced by getPtr(getKey(i))
