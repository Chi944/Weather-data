Detailed Description

DataLoader class Ś parses weather records from a single CSV file.

Opens a CSV file, detects column positions from the header row, and converts each data line into a WeatherRecord. Column order is determined dynamically so that differently-ordered CSV files are supported.

Error information is stored internally and retrieved via GetLastError(). This class contains no console output.
