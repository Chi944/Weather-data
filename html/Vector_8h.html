Detailed Description

Template Vector class a generic dynamic array with automatic resizing.

Provides a minimal but complete dynamic-array implementation templated on element type T. Internal storage is delegated to std::vector, which handles capacity doubling automatically on insertion, giving amortised O(1) append time.

All index access is bounds-unsafe by design (no exceptions thrown) for performance; callers are responsible for supplying valid indices.
