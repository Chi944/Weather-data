Write one month data row to the CSV file with stdev AND MAD (option 4).

Output format per A2 spec: MonthName, windMean(windStdDev, windMAD),tempMean(tempStdDev, tempMAD),solar

If a field has no data (hasWind/hasTemp/hasSolar = false) the field is left blank between its surrounding commas, e.g.: February, 4.5(3.1, 2.9),,200.3

Parameters
