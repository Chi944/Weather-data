void MenuHandler::PrintYear ( int &  year)
staticprivate

Static visitor passed to Bst<int>::inOrderTraversal as f1Typ.

Signature is void (*)(int&) to match Bst<T>::f1Typ exactly. Must be static so its address is a plain function pointer (not a member pointer), which is what Bst<T>::inOrderTraversal requires. Prints the year value followed by a space.

Parameters
yearYear value delivered by the BST traversal.
