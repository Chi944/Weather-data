Detailed Description

Unit tests for OutputHandler (Assignment 2) — Test Plan tests 1-12.

Strategy: redirect std::cout to a std::ostringstream before each call and inspect the captured string. CSV methods write to a temporary std::ostringstream-backed stream so no real files are created.

Tests cover all Assignment 2 methods: DisplaySPCCHeader, DisplaySPCC, DisplaySPCCNoData, WriteCSVRow with stdev+MAD, WriteCSVNoData, and the existing option-1/2/4 methods to confirm they still produce correct output.

Manual verification of expected strings: DisplayWindStats(3, 2020, 5.4, 2.5): "March 2020:", "Average speed: 5.4 km/h", "Sample stdev: 2.5"

DisplayTempForMonth(1, 15.0, 7.1): "January: average: 15.0 degrees C, stdev: 7.1"

DisplaySPCC(0.85, -0.12, 0.67): "S_T: 0.85", "S_R: -0.12", "T_R: 0.67" (2 d.p.)

WriteCSVRow month=5 (May), hasWind=true wind=3.6/0.0/0.0, hasTemp=true temp=20.0/0.0/0.0, hasSolar=true solar=0.1: "May, 3.6(0.0, 0.0),20.0(0.0, 0.0),0.1"
