Sample Pearson Correlation Coefficients for a given month across ALL loaded years.

Combinations computed:

  • S_T: Wind Speed vs Temperature (all records for the month)
  • S_R: Wind Speed vs Solar Radiation (records where SR >= 100 W/m2 only)
  • T_R: Temperature vs Solar Radiation (records where SR >= 100 W/m2 only)
Parameters
storeIndexed weather data store.
monthMonth number (1-12).
s_tOutput: sPCC for Wind Speed vs Temperature.
s_rOutput: sPCC for Wind Speed vs Solar Radiation.
t_rOutput: sPCC for Temperature vs Solar Radiation.
Returns
true if enough data existed to compute at least S_T.
